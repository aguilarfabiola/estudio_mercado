# Estudio de Mercado Inmobiliario - Talca

Dashboard web estatico para publicar en GitHub Pages.

## Archivos necesarios para publicar

- `index.html`
- `talca-analysis-data.js`
- `talca-permisos-data.js`
- `independencia-logo.webp`
- `.nojekyll`

La plataforma mantiene login de acceso por clave en el navegador. Para seguridad real con datos sensibles se requiere autenticacion de servidor.
